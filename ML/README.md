# Deprecated ML directory

Training scripts and metadata were consolidated under `dataset/`.

- **Training pipeline:** `dataset/pipeline/`
- **Raw data:** `dataset/data/raw/`
- **Processed pairs:** `dataset/data/processed/`
- **Model artifact:** `dataset/models/plagiarism_model_v3.pkl`

The duplicate Python scripts previously in `ML/dataset/` were moved to `dataset/archive/scripts/ml_legacy/`.

The duplicate submission trees under `ML/dataset/original/` and `ML/dataset/plagiarized/` are no longer maintained. Use `dataset/original/` and `dataset/plagiarized/`.
